Alguns modelos de banco de dados, comumente usados durante a historia:

- Modelo hierárquico:

	- Os dados são organizados de forma hierárquica, com conjuntos interconectados por meio de ligações;

	- Uma ligação representa uma relação entre dois tipos de registros: pai e filho;

	- O acesso aos dados é sempre unidirecional, a partir do pai ao filho;

	- Um esquema desse modelo é um diagrama de estrutura em árvore;


- Modelo em Rede: 

	- No modelo em redes os dados são organizados em tipos e ligações entre dois registros;

	- Tanto o esquema quanto ocorrencias de dados são visualizadas como garfo direcionado;

	- Não há restrição hierárquica;


- Modelo relacional:

	- Separamos os dados em entidades, que contém atributos.

	- As entidades se relacionam entre si e permitem que os dados sejam 
	armazenados e recuperados de forma rápida e segura.
