para criar um ambiente virtual é necessário ter o python 3.x e a biblioteca virtualenv

após ter esses dois componentes, abra a pasta cujo queira criar dentro dela esse ambiente virtual.
abra o terminal e coloque o terminal nessa pasta usando o cd /caminho/

após isso execute:

>>> python -m venv (nome da env que desejar)

para ativar seu ambiente virtual basta digitar no mesmo terminal:

>>> .\NomeDoSeuAmbiente\Scripts\Activate 

IMPORTANTE RESSALTAR QUE PARA ATIVAR, É REALMENTE NECESSÁRIO AS BARRAS INVERTIDAS.
