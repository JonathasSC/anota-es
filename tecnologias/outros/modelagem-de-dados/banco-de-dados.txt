Banco de Dados:

Um banco de dados é uma coleção organizada de dados.
Esses dados são organizados de modo a modelar aspectos do mundo real,
para que seja possivel de forma virtual a organização e a melhor utilização
dos dados.

Um banco de dados é composto por diversões objetos, como:

- Tabelas;
- esquemas; 
- visões;
- consultas; 
- relátorios; 
- procedimentos;
- triggers; 

entre outros.
